카카오맵 API 소스로 변경이 필요할 것입니다.

아래는 카카오맵 API로 주소를 검색하는 소스입니다.

$headers = array(“Authorization: KakaoAK “.$daum_api_key);
$opts = array(
CURLOPT_URL => “https://dapi.kakao.com/v2/local/search/address.json?query=".urlencode($address), CURLOPT_RETURNTRANSFER => true,
CURLOPT_HTTPGET => true,
CURLOPT_SSL_VERIFYPEER => false,
CURLOPT_SSL_VERIFYHOST => 0,
CURLOPT_SSLVERSION => true,
CURLOPT_HEADER => false,
CURLOPT_HTTPHEADER => $headers
);

$curl_session = curl_init();
curl_setopt_array($curl_session, $opts);
$return_data = curl_exec($curl_session);
curl_close($curl_session);

$json_decode = json_decode($return_data, true);

return $json_decode;

확인해보시고 개발자님들과 상의해서 올해말까지 작업 진행하셔야 할 것입니다.
