오늘은 RxSwift 소스를 마음대로 돌려볼 수 있는 Playground 를 만드는 방법을 알아보겠습니다.

RxSwift Playground 만드는 방법

  1. xcode 에서 Single View App을 생성합니다.

  1. Playground 를 추가합니다.

아래와 같이 Playground 가 추가되어 정상적으로 동작합니다.

  1. xcode 를 닫습니다.

  2. 터미널을 이용해 위에서 만든 프로젝트의 경로로 이동합니다.

  1. 터미널에서 pod init 을 실행합니다.

  2. 터미널에서 vi Podfile 로 Podfile 을 엽니다.


열면 위와 같을 것입니다.

  1. 터미널에서 열린 파일 가장 밑에 줄 end 위에 아래 코드를 추가하고 저장합니다.

  1. 터미널에서 pod install 을 실행합니다.

  2. 정상적으로 설치됬다면 RxSwiftPlayground.xcworkspace 파일이 생성되었을 것입니다. 실행합니다.

  3. Playground 에 RxSwift 를 import 하고 예제 코드를 돌려보시면 됩니다.

오류가 나올 때 해결 방법

위와 같은 오류가 발생하시는 분들이 많을 것입니다.

저 또한 매번 이런 오류가 발생했는데요.

해결 방법을 알려드리겠습니다.

  1. import RxSwift 를 지웁니다.
  2. 아래 부분에서 실행버튼을 눌러 오류를 발생시킵니다.
  3. 다시 improt RxSwift 를 작성합니다.
  4. 메뉴바에서 Product-Build 를 클릭하여 다시 빌드합니다.
  5. 아래 부분에서 실행버튼을 클릭해보면 정상적으로 옵저버블이 생성되는 것을 볼 수 있습니다.
